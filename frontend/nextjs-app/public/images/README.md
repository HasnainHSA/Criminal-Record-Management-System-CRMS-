# Images Folder

## Required Images for Sindh Police CRMS:

1. **logo.png** - Sindh Police logo (place here)
2. **badge.png** - Police badge icon (place here)
3. **hero-bg.jpg** - Background image for hero section (optional)

## Image Specifications:

- **logo.png**: 200x200px, transparent background
- **badge.png**: 100x100px, transparent background
- **hero-bg.jpg**: 1920x1080px (optional)

## How to Add Images:

1. Copy your Sindh Police logo as `logo.png`
2. Copy police badge as `badge.png`
3. Place them in this folder: `frontend/nextjs-app/public/images/`

The app will automatically use these images.
